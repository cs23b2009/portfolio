# Dasapathi Indra Kumar - Portfolio & Resume

This is my official developer portfolio and resume website.

## Overview
This project showcases my skills, experience, and projects as a Full Stack Developer. It is built with Next.js, Tailwind CSS, and TypeScript.

## Features
- **Resume**: View and download my resume directly from the website.
- **Projects**: Detailed showcase of my technical projects like "Video Calling App" and "Job Portal".
- **Experience**: A timeline of my professional journey and internships.
- **Skills**: A comprehensive list of my technical skills and technologies.
- **Contact**: Easy way to get in touch with me.

## Tech Stack
- **Framework**: Next.js
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **Deployment**: Vercel
